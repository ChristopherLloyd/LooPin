The Loop Atlas

Welcome to the loop atlas! This page contains an exhaustive catalog of generic isotopy classes of irreducible, indecomposable loops in the unoriented sphere, by number of complementary regions. The number of such loops matches sequence A264759 on oeis.org. The loops were generated with plantri and drawn with SnapPy. The title is a nod to the knot atlas. Click on each loop to see a larger view.
